## Claude Local Operating Notes

Prefer the local operating model below when working in this repository:

- Treat this project as a published-output consumer, not the primary wiki authoring source.
- Before changing content structure, metadata rules, or local agent surface, write or update a short plan first.
- Before completion, verify the changed scope explicitly.
- Keep compatibility with `content/posts` publishing contracts.
- When writing or revising posts, follow `docs/content-seo-guide.md` so future content also reflects the repository's SEO writing rules.
- When equivalent authoring rules exist upstream in `ai-survival-log`, keep them aligned with this repository so blog-writing and series-writing rules do not diverge across repos.
- Check for consistency across `README.md`, `AGENTS.md`, `CLAUDE.md`, `.codex/*`, and `docs/content-contract.md` when changing operational guidance.
- Request review when changing content contracts, series rules, or local operating surfaces.
